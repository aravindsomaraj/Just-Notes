### 3 Rules:
1. do action; stmt  
        $\equiv$ action >> do stmt
2. do x <- action; stmt  
        $\equiv$ action >>= $\lambda$x -> do stmt
3. do action $\equiv$ action
   
```Haskell
do putStr "Give an integer:"
    do inp <- get
    print(inp+1)

-- Unsugared version:
putStr "Give an integer"
    >> (get >>= \inp -> print(inp+1))
```
